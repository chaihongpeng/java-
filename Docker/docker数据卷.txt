VOlume容器间的数据共享
场景使用，部署tomcat的集群是,web文件都是相同的，如果修改了web文件，在没有容器的数据共享功能时，就要对每一个容器进行更新
数据共享方案，在宿主机上开辟一块空间，通过docker配置多个容器之间读取同一份问价
数据共享的实现办法
docker run -v <主机路径>:<容器路径> <镜像名>
这样做的一个缺点，每次创建容器时书写挂载路径，有任意大小写或其他错误就会造成挂载失败

第二个办法，通过创建共享容器来完成数据的挂载
第一步创建容器
docker create --name <容器名> -v <宿主机路径>:<容器路径> <镜像名>
共享容器挂载点
docker run --volumes-from <被指定的共享容器名> <镜像名>
这样新创建的容器会根据被指定的原有容器寻找挂载点
