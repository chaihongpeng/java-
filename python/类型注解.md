python3.5版本引入类型注解

## 变量类型注解

```
变量: 类型
```

```python
var_1: int = 10
var_2: float = 3.1415926
var_3: bool = True
var_4: str = "hello world"
stu: Student = Student()

my_list: list[int] = [1, 2, 3]
my_tuple: tulpe[str, int, bool] = ("xiaoming", 666, True)
my_set: set[int] = {1, 2, 3}
my_dict: dict[str, int] = {"xiaoming", 666}
```
