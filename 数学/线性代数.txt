向量在数学中的不同:
    向量不同于乘法交换律: AB != BA
    向量中的零: 0:数字零;0:矩阵零;0:零向量
    分母: 矩阵不能作为分母
行列式:
排列: 由1,,2...,n组成的一个有序数组就叫一个n级排列;中间不能缺数，数字位置可以自由交换，n级排列有n!种（n的阶乘种）
逆序: 大数排在比较小的数的前面
逆序数: 逆序的总数就是逆序数
标准排列: 1,2,3,4一直到n,没有逆序的排列叫做n级标准排列(自然排列)
对换: 给出一个排列，交换其中两个数的位置
排列的奇偶性:
n阶行列式:
    - 按行展开定义: 行标标准排列，列表按就行，偶加基减
三角行列式: 对角线一侧全是零



矩阵:
	m行 n列 m*n矩阵
方阵:
行向量:
	只有一行的矩阵,m=1
列向量:
	只有一列的矩阵,n=1
两个矩阵相等:
	行列数相同,同位置的值相同
零矩阵o:
	矩阵上的数值全是零
单位矩阵E:
	右斜线上全为1,其余位置全为零
对角矩阵:
	只有右斜线上数据可以不为0
矩阵的加减:
	- 只有m和n都相同的矩阵能做加减,同坐标位置相加;
	- 满足交换律和结合律;A + B = B + A;(A + B) + C = A + (B + C);
矩阵的数乘:
	- λA就是在A的每一个向量都乘λ;
	- (λμ)A = λ(μA);
	- λ(A + B) = λA + λB;
矩阵的乘法:
	- 条件:第一个矩阵的列数等于第二个矩阵的行数
	- m行矩阵 * n列矩阵 = m*n矩阵
	- 矩阵的乘法不满足交换律
	- 满足结合律和分配率(AB)C = A(BC);λ(AB) = (λA)B = A(λB)
	- A(B+C) = AB + AC
矩阵的迹tr
	- 矩阵的迹就是左斜对角线求和,只有方阵才可以求迹
	- tr(AB) = tr(BA)
矩阵的转置
	- A = A转置,则A为对称矩阵
